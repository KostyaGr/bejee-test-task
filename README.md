# Приложение для ведения списка задач
### Backend stack:
0. PostgreSQL
0. Node.JS
1. Express
2. Sequelize
### Frontend stack:
0. React.JS
1. Redux Toolkit + RTK query
2. Material UI
3. React-hook-form
4. react-toastify

Для того чтобы запустить приложение нужно [установить docker](https://docs.docker.com/engine/install/ubuntu/) с аддоном docker-compose и запустить команду `docker compose up -d` в папке с проектом.
